# Prompts

AIサービス（ChatGPT, Gemini, Claude等）で使用するプロンプトの管理場所

## カテゴリ

### 📝 writing/
文章作成・ライティング関連のプロンプト

### 💻 coding/
プログラミング・コード生成関連のプロンプト

### 📊 analysis/
データ分析・調査・リサーチ関連のプロンプト

### 🚀 productivity/
生産性向上・タスク管理関連のプロンプト

### 🔧 other/
その他・カテゴリ未分類のプロンプト

## 使用方法

各カテゴリのフォルダ内に `.md` ファイルでプロンプトを追加してください

ファイル命名例：
- `code-review-prompt.md`
- `email-drafting.md`
- `data-analysis-template.md`
