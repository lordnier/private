ソクラテス式ver1.01

【1. 明確化の質問】
1. 今の具体的なゴールは何か？
2. 本当に解きたい問題は？
3. 制約がなければ何を望む？

【2. 前提調査の質問】
1. 今の前提は何か？
2. それは事実？解釈？
3. 他にどんな世界解釈がある？
4. Whyを5回繰り返すと？
5. 逆にすると？

【3. 証拠の質問】
1. 実例・類似構造は？
2. 他分野で似ているものは？
3. この問題を一段抽象化すると何問題？
4. 原因ではなく“システム構造”で見ると？

【4. 視点変更の質問】
1. 他にどんな可能性がある？
2. 他人ならどう考える？
3. SCAMPERで崩すと？
4. 制約ゼロなら？
5. 最悪化すると？
※この段階では評価禁止

【5. 影響調査の質問】
1. 何が最も本質的か？
2. 長期で効くのは？
3. レバレッジが高いのは？
4. 可逆性が高いのは？
5. 一番少ない労力で効果が大きいのは？

【6. 意思決定の質問】
1. ゴールから逆算するとどんな手順になるか？
2. 一番リスクが低い実験は？
3. どこが最大のボトルネックになりそうか？
4. どこが最大のボトルネックになりそうか？
5. やめる条件・続ける条件は？
